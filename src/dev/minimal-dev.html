React-LLM-UI Development Mode

This is a minimal development environment for testing the UI modules system.

To use the full development environment, you'll need to build the library first with npm run build.

Getting Started

Follow these steps to set up the dev environment:

  1. Create a new React application or use an existing one
  2. Import the ModuleProvider and desired modules
  3. Register the modules and wrap your application with ModuleProvider

Example Usage


import React from 'react';
import { 
  ModuleProvider, 
  registerDefaultModules,
  ChatOverlayModule,
  DebugPanelModule
} from 'react-llm-ui';

// Register default modules
registerDefaultModules();

const App = () => {
  return (
    <ModuleProvider
      defaultVisibility={{
        [ChatOverlayModule.id]: true,
        [DebugPanelModule.id]: false
      }}
      enableControlPanel={true}
      controlPanelPosition="top-right"
    >
      <YourApplication />
    </ModuleProvider>
  );
};

Quick Integration

Add this module to any existing React application:

  1. Install the package: npm install @sschepis/react-copilot
  2. Add the provider to your app: import { ModuleProvider, registerDefaultModules } from '@sschepis/react-copilot'
  3. Register modules and wrap your app with ModuleProvider

For a more detailed setup, check the documentation.
