React-LLM-UI Development Mode

This is a simple development environment to view the UI modules without building the entire library.

Modules View

The following modules are simulated here:

  • Chat Overlay - A basic chat interface for LLM interactions
  • Debug Panel - Tools for debugging and component inspection
  • UI Control Panel - Controls for toggling module visibility

Chat Overlay

User: Hello, how can I use the UI modules?
